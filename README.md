# BBMKU M.A. Political Science Dashboard

This is the dual-language interactive study dashboard for BBMKU M.A. Political Science & UGC NET Preparation.

## Project Structure
- `index.html`: The main dashboard file containing the dual-language content.
- `css/style.css`: Contains all styling, UI transitions, print configurations (A4 perfect margin and page-break logic).
- `js/app.js`: Contains logic for Tab Switching and Language Toggle (English/Hindi).
- `assets/`: Folder for future images or logos.

## How to Run locally
You can run this project in any standard web environment, including:
1. Double-clicking `index.html` directly in your browser.
2. Using VS Code "Live Server" extension.
3. Using Python's built-in server: Run `python -m http.server 8000` in this directory.

## Deployment
Since it's pure HTML/CSS/JS, you can easily host this on:
- Firebase Hosting (just run `firebase deploy`)
- GitHub Pages
- Netlify / Vercel
